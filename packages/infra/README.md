# @starter/infra-components

AWS CDK constructs for the Starter Platform infrastructure.

## Installation

Ensure your local `.npmrc` is configured to point `@starter` to GitHub Packages.

```bash
pnpm install @starter/infra-components
```

## Documentation

- [Developer Setup Playbook](https://internal-wiki.atlassian.net/wiki/spaces/STARTER/pages/4107206689/Starter+Starter+Digital+Architect+and+Developer+Getting+Started+Guide) - Instructions for configuring your local environment (PAT, .npmrc).
