# @starter/ui-components

Angular UI package for the Starter Platform.

## Components

| Component  | Selector        | Description                                              |
| ---------- | --------------- | -------------------------------------------------------- |
| `HomePage` | `starter-home-page` | Home page layout with hero banner and content projection |

## Installation

Ensure your local `.npmrc` is configured to point `@starter` to GitHub Packages.

```bash
pnpm install @starter/ui-components
```
